因子工程

QUANT 量化因子

  • EPS、ROE、D/E、FCF、净利润增速
  • 保持原始数值，不标准化
  • 输出: *_quant_quarterly.csv

QUAL / LLM 定性因子 (LLM 驱动)

  • 5步子代理流程: 商业模型 → 财务概览 → 因子选择 → 证据矩阵 → 评分 (1-10)
  • 简化4因子: 市场潜力 / 定价权 / 品牌力 / 管理质量
  • 支持 Gemini / OpenAI / Mock 多 LLM 后端
  • 输出: *_qual_quarterly_scores.csv + rationale

MACRO 宏观因子

  • 存款利率 → PE 锚定
  • 季度频率对齐
  • 输出: macro_{CC}_{series}.csv
▼ ▼ ▼
参数合成

RULES 规则合成器

  • 定性评分 → 偏离指数 (score - 5.5)
  • 调节增长率 ±1%、WACC ±0.5%
  • 调节估值倍数 ±15%
  • 公式: adj = base × (1 + α × dev)

ROUTER 估值核路由

  • LLM Turn 1: 分析商业模型
  • 自动选择最优估值方法
  • DCF / P/S / PB-ROE 三选一

MLP 神经合成 (Phase 2)

  • 可学习参数映射
  • 替代规则合成器
▼ ▼ ▼
估值计算

DCF 现金流折现

  • FCF + 增长率 + WACC
  • 终值 = FCF×(1+g) / (WACC-g)
  • → 内在价值/股

P/S 市销率

  • 每股销售额 × 目标 P/S
  • → 隐含价格

PB-ROE 市净率

  • 每股净资产 × Justified P/B
  • ROE vs 权益成本

PE-Macro 市盈率(宏观)

  • 利率 → 基础PE (反比)
  • 定性乘数: [0.5, 1.2]
  • 最终PE ∈ [10, 24]

输出层

  • 季度 Fi → 日频前向填充
  • 120日移动均线平滑
  • Vanilla Fi + 定性调节 Fi 双线
  • 输出: fi_quarterly.csv / intrinsic_quarterly.csv

VIZ 可视化 Dashboard

  • 股价 Pt vs 内在价值 Fi 叠加图
  • 偏离度 = (Pt - Fi) / Fi
  • 因子时间线 + 审计追踪
  • 输出: *_dashboard.png / overlay.png

  • 输出: *_meta.json
